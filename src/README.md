# 📺 YouTube Clone – React App

A YouTube-like video streaming web application built using **React.js** and **Tailwind CSS**.
This project focuses on UI, component structure, routing, and API-based data rendering.

---

## 🚀 Features

- 🏠 Home page with video listings
- 🔍 Search videos
- 📺 Watch video page
- 💬 Comments section (UI based)
- 🎞 Related videos
- ⚡ Shimmer UI while loading
- 🔀 Routing using React Router DOM
- 📱 Fully responsive (Tailwind CSS)

---

## 🛠 Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **HTML**
- **Parcel **
- **YouTube Public API**
