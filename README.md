# YouTube React App

A fully functional **YouTube clone** built with modern web technologies.  
This project allows users to search for videos, view results, and watch videos in a responsive and interactive interface.

## 🚀 Features
- Search for videos by keyword  
- Responsive layout for **mobile and desktop**  
- Video player integration  
- **Redux** state management for global state  
- Styled with **Tailwind CSS** for clean UI  
- Built with **React.js** and **JavaScript**  

## 🛠 Technologies Used
- **React.js** – Frontend library  
- **Redux** – State management  
- **JavaScript (ES6+)** – Application logic & API handling  
- **HTML5 & CSS3** – Structure & styling  
- **Tailwind CSS** – Utility-first styling framework  
- **REST APIs** – YouTube Data API (or mock API) for fetching videos  

## 📁 Project Structure
/src
/components # Reusable components (VideoCard, SearchBar, Navbar)
/pages # Page components (Home, VideoDetails)
/redux # Redux store and slices
App.js # Main app component
index.js # Entry point
